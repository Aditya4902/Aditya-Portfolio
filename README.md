## How to Run

**Option 1 — Open directly (no setup)**
```
Open index.html in any browser
```

**Option 2 — Local server**
```bash
npm install
node server.js
# Open http://localhost:3000
```

**Option 3 — Dev mode**
```bash
npm run dev
```
